# Sistem Manajemen Pesantren Al-Bisri

Sistem manajemen untuk LKSA (Lembaga Kesejahteraan Sosial Anak) Pesantren Al-Bisri.

## Quick Start

```bash
npm install
npm run dev
```

Server akan berjalan di **http://localhost:8080**
